=== Feature 38: O64 Liveness-Aware i8 Spill/Reload Lowering ===

Demonstrates O64 Shape B Row 3 firing at a reload site where both HL and
A are live but a spare GPR (B) is dead after the reload. O64 replaces
the classical PUSH HL / LXI HL / MOV r,M / POP HL sequence (4 insns,
48cc, 6B) with MOV B,A / LDA / MOV r,A / MOV A,B (4 insns, 40cc, 5B),
saving 8cc and 1 byte per occurrence.

----------------------------------------------------------------
1. Test case (v6llvmc.c = c8080.c, shared)
----------------------------------------------------------------

__attribute__((leaf)) extern unsigned char op(unsigned char x);
__attribute__((leaf)) extern void use5(unsigned char, unsigned char,
                                       unsigned char, unsigned char,
                                       unsigned char);

unsigned char many_i8(unsigned char a, unsigned char b, unsigned char c,
                      unsigned char d, unsigned char e) {
    unsigned char x1 = op(a);
    unsigned char x2 = op(b);
    unsigned char x3 = op(c);
    unsigned char x4 = op(d);
    unsigned char x5 = op(e);
    use5(x1, x2, x3, x4, x5);
    return (unsigned char)(x1 ^ x2 ^ x3 ^ x4 ^ x5);
}

int main(int argc, char **argv) {
    many_i8(0x11, 0x22, 0x33, 0x44, 0x55);
    return 0;
}

----------------------------------------------------------------
4. Notes
----------------------------------------------------------------

* c8080's main is smaller (30B vs 69B) and faster (160cc vs 378cc)
  because c8080 uses static-stack globals for EVERY argument and
  local, with zero register allocation overhead. v6llvmc still pays
  register-homing costs but O64 narrows the gap on every i8 reload
  site that meets the Row 3 precondition.

* many_i8 is dominated by 5x (LDA slot + CALL + STA slot) sequences;
  no Shape B/C sites appear after RA in this function with the
  current schedule. O64 leaves this function unchanged.

* The O64 ladder also covers Shape C (H/L sites) and Shape B Row 2
  (A dead), which this test does not exercise. Coverage for those
  rows is provided by the O64 lit test
  `llvm/test/CodeGen/V6C/spill-reload-i8-static-shapes.ll`.
