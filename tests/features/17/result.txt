== O38: XRA+CMP i8 Zero-Test Peephole ==

=== C Test Case ===

extern unsigned char bar(unsigned char x);

// Primary pattern: two consecutive zero-tests.  Second one triggers
// MOV A,r; ORA A; JZ → XRA A; CMP r; JZ.  Since XRA A sets A=0,
// LoadImmCombine eliminates downstream MVI A,0.
unsigned char test_two_cond_tailcall(unsigned char x, unsigned char y) {
    if (x) return bar(x);
    if (y) return bar(x);   // x==0 here; bar(0) needs A=0
    return 0;
}

// Negative: single zero-test, nothing to transform.
unsigned char test_simple_zero_check(unsigned char val) {
    if (val) return 1;
    return 0;
}

// Negative: result in register, not zero pattern.
unsigned char test_nz_branch(unsigned char a, unsigned char b) {
    if (a) return b;
    return 0;
}

=== Improvement Summary ===

Primary pattern (test_two_cond_tailcall):
  Bytes saved: 7B (20B → 13B)
  Cycles saved: 16cc (60cc → 44cc)
  Eliminated: MOV A,E (8cc,1B), 2× MVI A,0 (16cc,4B), JZ+RET → RZ (12cc,2B)
  Gained: XRA A (4cc,1B), CMP E (4cc,1B)

vs c8080:
  test_two_cond_tailcall:  v6llvmc 13B vs c8080 30B (17B smaller)
  test_nz_branch:          v6llvmc 7B vs c8080 19B (12B smaller)

Transformation chain:
  1. V6CPeephole: MOV A,E; ORA A; JZ → XRA A; CMP E; JZ
  2. V6CBranchOpt: JZ + RET → RZ (conditional return folding)
  3. V6CLoadImmCombine: XRA A seeds A=0 → MVI A,0 eliminated on both paths
