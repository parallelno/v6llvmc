    if (x < 1000) result = 1;
}

    if (x >= 1000) result = 2;

== Summary ==

O24: MVI+SUB/SBB Immediate Ordering Comparison

Per ordering comparison instance:
  Old (LXI+MOV+SUB+MOV+SBB):  36cc, 7B, uses DE register pair
  New (MVI+SUB+MVI+SBB):       24cc, 6B, no register pair used
  Savings:                      12cc, 1B, +1 register pair freed

Applied to: BR_CC (branch) and SELECT_CC (ternary) with i16 ordering
conditions (ULT, UGE, UGT, ULE, SLT, SGE) and constant operand.

Comparison vs c8080:
  c8080 unsigned: 32cc, 7B (DAD trick, cheaper but limited to unsigned)
  c8080 signed: 72cc, 10B (library CALL + SUB/SBB, much more expensive)
  v6llvmc new: 24cc, 6B (both unsigned and signed, fastest)
