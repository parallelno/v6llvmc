== O38: XRA+CMP i8 Zero-Test Peephole ==

=== C Test Case ===

extern unsigned char bar(unsigned char x);

// Primary pattern: two consecutive zero-tests.  Second one triggers
// MOV A,r; ORA A; JZ → XRA A; CMP r; JZ.  Since XRA A sets A=0,
// LoadImmCombine eliminates downstream MVI A,0.
unsigned char test_two_cond_tailcall(unsigned char x, unsigned char y) {
    if (x) return bar(x);
    if (y) return bar(x);   // x==0 here; bar(0) needs A=0
    return 0;
}

// Negative: single zero-test, nothing to transform.
unsigned char test_simple_zero_check(unsigned char val) {
    if (val) return 1;
    return 0;
}

// Negative: result in register, not zero pattern.
unsigned char test_nz_branch(unsigned char a, unsigned char b) {
    if (a) return b;
    return 0;
}

=== v6llvmc ASM (before O38 — baseline) ===

test_two_cond_tailcall:         ; 13B (second-test path)
    ORA  A                      ; 4cc, 1B — first test (x==0)
    JZ   .LBB0_1               ; 12cc, 3B
    JMP  bar                    ; 12cc, 3B
.LBB0_1:
    MOV  A, E                   ; 8cc, 1B — copy y to A
    ORA  A                      ; 4cc, 1B — zero-test y
    JZ   .LBB0_4               ; 12cc, 3B
    MVI  A, 0                   ; 8cc, 2B — A was clobbered
    JMP  bar                    ; 12cc, 3B
.LBB0_4:
    MVI  A, 0                   ; 8cc, 2B — return 0
    RET                         ; 12cc, 1B

=== v6llvmc ASM (after O38) ===

test_two_cond_tailcall:         ; 6B (second-test path)
    ORA  A                      ; 4cc, 1B — first test (x==0)
    JZ   .LBB0_1               ; 12cc, 3B
    JMP  bar                    ; 12cc, 3B
.LBB0_1:
    XRA  A                      ; 4cc, 1B — sets A=0, sets Z from E
    CMP  E                      ; 4cc, 1B — compares A(=0) with E
    RZ                          ; 12cc, 1B — conditional return (JZ+RET folded)
    JMP  bar                    ; 12cc, 3B — A already 0 for bar(0)

=== Improvement Summary ===

Primary pattern (test_two_cond_tailcall, second-test path):
  Bytes saved: 7B (13B → 6B)
  Cycles saved: 24cc on NZ path, 16cc on Z path (worst case)
  Eliminated: MOV A,E (8cc,1B), 2× MVI A,0 (16cc,4B)
  Gained: XRA A (4cc,1B), CMP E (4cc,1B) — net +2B +8cc for compare
  JZ+RET folded to RZ: saves 2B, 12cc (conditional return O23)

Transformation chain:
  1. V6CPeephole: MOV A,E; ORA A; JZ → XRA A; CMP E; JZ
     (condition: A is dead after branch OR fallthrough needs A=0)
  2. V6CBranchOpt: JZ + RET → RZ (conditional return folding)
  3. V6CLoadImmCombine: XRA A seeds A=0 → downstream MVI A,0
     eliminated on both paths via branch-seed propagation

Pipeline ordering:
  Peephole runs before LoadImmCombine so XRA A is visible as A=0
  source. LoadImmCombine's seed propagation (Pattern D) recognizes
  XRA A; CMP r; JZ/JNZ and propagates A=0 to all successors.

New LoadImmCombine enhancements:
  - XRA A tracking: A=0 for value-tracking (was treated as ALU clobber)
  - Pattern D before ZeroProvenPath gate: seeds A=0 unconditionally
  - Try 0 elimination: if register already holds target value, erase MVI

test_simple_zero_check: unchanged (no MOV+ORA pattern)
test_nz_branch: unchanged (no MOV+ORA pattern)
