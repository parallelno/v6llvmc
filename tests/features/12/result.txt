=== C test case ===

int select_second(int a, int b, int c) {
    if (a) return b;
    return c;
}

int select_on_zero(int x, int a, int b) {
    if (x == 0) return a;
    return b;
}

int select_nonzero(int x, int a, int b) {
    if (x != 0) return a;
    return b;

Key improvement: O34 eliminates LXI rp,0 (3B+12cc) and SUB/SBB comparison,
replaces with MOV A,H; ORA L zero-test (2B+12cc). Freed register pair also
eliminates all spills in these functions — no frame setup/teardown needed.

v6llvmc is register-based (args in HL/DE/BC) while c8080 uses static stack
(args via memory loads/stores), giving v6llvmc a structural advantage for
small functions with few arguments.
